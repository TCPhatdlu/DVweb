Tên dự án: Ebook2LateX Web Service

Mô tả

Ebook2LateX là hệ thống dịch vụ web hỗ trợ trích xuất công thức toán học từ tài liệu PDF, chuyển đổi sang định dạng LaTeX và hiển thị trên giao diện web. Hệ thống được xây dựng theo kiến trúc client-server nhằm hỗ trợ học tập, nghiên cứu và số hóa tài liệu toán học.

Công nghệ sử dụng

Python
FastAPI
SQLAlchemy
PostgreSQL
Docker
React

Chức năng chính

Đọc tài liệu PDF
Nhận diện công thức toán học
Chuyển đổi công thức sang LaTeX
Lưu trữ dữ liệu lên cơ sở dữ liệu
Cung cấp REST API
Hiển thị kết quả trên giao diện web
